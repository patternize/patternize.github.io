#Pros/Cons

  • Advantage: - calls to get and set take constant time
  • Disadvantage: - insertion and removal are expensive unless changes are made at the end of arraylist - inefficient for search - notation of capacity (size of underlying array) - insertion has to be done manually, there is no method written for insertion
