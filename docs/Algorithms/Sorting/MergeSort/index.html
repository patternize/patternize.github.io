Merge Sort

#Idea

  • Use divide and conquer paradigm

#Procedure

  • divide array into two subarrays
  • sort each subarray
  • merge sorted subarrays into one

#Time and Space Analysis

  • Time O(nlogn)
  • Space O(nlogn)
